## 📊 Project Title

## Sales Performance Analytics Dashboard – Power BI

# 📚 Acknowledgement  
This project is a practice-based implementation inspired by a Power BI tutorial.
Special thanks to the original creator for guidance:
- FinTechy – Power BI Dashboard Tutorial  
- https://www.youtube.com/watch?v=YLU9sJuWRJ0

# 🎯 Objective
To design an interactive dashboard that analyzes sales performance, revenue trends, and customer behavior, enabling stakeholders to make data-driven business decisions.

# 📖 Project Description
This Power BI dashboard provides a comprehensive view of sales data, focusing on key metrics such as revenue, profit, and order trends. It allows users to explore performance across regions, products, and time periods using dynamic and interactive visuals.

# 🔧 Technologies Used
Power BI Desktop; DAX (Data Analysis Expressions); Power Query (ETL); CSV / Excel Dataset

# 🚀 Key Features
📊 KPI Cards: Total Sales, Profit, Orders
📅 Time-based analysis (monthly/yearly trends)
🌍 Region-wise and category-wise performance
🎛️ Interactive slicers (Date, Region, Category)
🔄 Cross-filtering across visuals
🔖 Bookmark-based navigation (if used)

# 📊 Business Calculations
Total Sales;  Total Profit;  Profit Margin;  Total Orders; Sales Growth (time-based)

# 📈 Visuals Used
KPI Cards: Line Chart (Sales Trend); Bar Chart (Category/Region); Pie/Donut Chart (Distribution); Slicers (Filters)

# 🔧 How to Run
Open .pbix file in Power BI Desktop
Load/refresh dataset
Interact using slicers and visuals
